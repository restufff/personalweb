## About The Project

Simple fetching api from [TheMealDB](https://www.themealdb.com/).
So this is the first application I made when practicing about ES6 especially fetching API, in this project i was using webpack to minify production deploying. There are 2 features, first you can search some recipe with the food name, second feature that you can search randomize recipe
